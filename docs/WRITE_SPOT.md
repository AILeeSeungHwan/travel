# WRITE_SPOT.md — 스팟 포스트 작성 규칙

## 분량 / 구조

- 본문 1,500자 이상
- 갤러리 3~5장 (모두 라이선스 메타 포함)
- **이미지가 없으면 발행 금지**

## 필수 섹션 (순서)

1. image — 갤러리 1장
2. intro — 위치·입장료·소요시간·베스트 시즌
3. h2 "이곳은 어떤 곳" (Level C 큐레이션)
4. h2 "하이라이트" (3~5개)
5. h2 "방문 팁"
6. h2 "가는 방법"
7. h2 "주변 함께 가볼 곳"
8. h2 "이 스팟 근처 추천 호텔"
9. warning (해당 시) "주의사항"
10. faq
11. sources
12. disclaimer
