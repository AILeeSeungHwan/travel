# WRITE_TOOL.md — 계산기 포스트 작성 규칙

## 분량 / 구조

- 위젯 + 본문 1,500자 이상

## 필수 섹션

1. intro — 무엇을 계산하는지·결과의 한계
2. (위젯 영역 — 페이지가 자체 처리)
3. h2 "어떻게 사용하나요"
4. h2 "결과 해석"
5. h2 "활용 팁"
6. h2 "계산기의 한계 / 흔한 오해"
7. faq
8. sources
9. disclaimer

## 추천 위젯 (data/tools.js 의 slug 와 일치)

- `travel-budget` — 항공+호텔+식비+교통+액티비티 합산
- `flight-time` — 출발지·도착지 자동 계산
- `currency` — 한국은행 ECOS API
- `luggage-weight` — 항공사별 수하물 한도
- `visa-checker` — 한국 여권 기준 비자 자동 분류 (Level A)
- `itinerary-builder` — 동선 자동 계산
