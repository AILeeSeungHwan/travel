# WRITE_COUNTRY.md — 국가 포스트 작성 규칙

## 분량 / 구조

- 본문 2,500자 이상
- 인트로 2~3단락
- H2 8~10개, 각 H2 아래 본문 200자+

## 필수 섹션 (순서)

1. intro — 베스트 시즌 + 비자 + 통화 + 시간대 한 줄 요약
2. info 또는 risk — 비자/안전 (Level A면 risk, B/C면 info)
3. h2 "이 나라는 어떤 곳" — Level C 큐레이션
4. h2 "지역별로 가보기" — region 카드 그리드
5. h2 "인기 여행 테마"
6. h2 "추천 호텔 BEST"
7. h2 "여행 가이드 (비자·교통·통화)"
8. h2 "여행 전 알아둘 것" — 안전·세관·문화
9. faq 5~8개
10. sources 2개 이상
11. disclaimer

## 출처 (Level A는 2개 필수)

- 외교부 해외안전여행 (0404.go.kr) — 비자/안전
- 한국관광공사 또는 해당국 관광청
- UNESCO (해당 시)
