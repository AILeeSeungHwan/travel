# WRITE_GUIDE.md — 가이드 포스트 작성 규칙

## 분량 / 구조

- 본문 3,500자 이상 (Level A 비자/안전은 정확성 우선)
- H2 10개 이상

## Level A (비자·안전·여행자보험·여행경보) — 가장 엄격

1. risk box "출국 전 재확인 필수"
2. intro
3. h2 "핵심 요약"
4. h2 "필요 서류"
5. h2 "신청 절차" — HowTo 스키마용 step 배열
6. h2 "비용"
7. h2 "유효기간 / 갱신"
8. h2 "거부 사유 / 대응"
9. h2 "한국 여권 vs 다른 국가"
10. faq
11. sources **2개 이상 필수** — 외교부·해당국 대사관·항공사
12. disclaimer

## Level B (교통·통화·환전·면세)

1. intro
2. h2 핵심 요약
3. h2 옵션 비교 (트래블카드 vs 환전 vs ATM 등)
4. h2 절약 팁
5. h2 변동 요소 (환율·수수료)
6. faq
7. sources
8. disclaimer

## Level C (언어·에티켓·짐 싸기)

자유 형식, 출처 권장.
