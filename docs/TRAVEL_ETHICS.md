# TRAVEL_ETHICS.md — 여행 도메인 윤리·라이선스·면책

> 본 사이트의 모든 콘텐츠는 본 문서의 규칙을 따른다. 위반 시 발견 즉시 수정·삭제.

| 등급 | 범위 | 출처 요건 | 면책 |
|---|---|---|---|
| **A** | 비자·출입국·여행자보험·외교부 여행경보·환율 정책 | 외교부·해당국 대사관·한국은행 ECOS 등 공식 2개 이상 | "출국 전 외교부 재확인" 고정 |

| **C** | 여행 후기·추천·문화 | 자체 큐레이션 가능, 출처 권장 | 일반 면책 |

## 2. 이미지 라이선스 (가장 중요)

### 2-1. 허용 출처 (이 5개 외 사용 금지)

| 출처 | 라이선스 | 활용 | 필수 표기 |
|---|---|---|---|
| 한국관광공사 TourAPI 4.0 | 공공누리 1유형 | 국내 관광지·문화유산·축제 | "출처: 한국관광공사" |
| Wikimedia Commons | CC BY / CC BY-SA / Public Domain | 해외 도시·랜드마크 | 작가명 + 라이선스 |
| Unsplash / Pexels API | Unsplash·Pexels License | 분위기·배경 | "Photo by {작가}" |
| 호텔스컴바인 API hot-link | 파트너 약관 | 호텔 객실/외관 | "via HotelsCombined" |
| 자체 OG | 자체 자산 | 포스트 메인 | 표기 불필요 |

### 2-2. 금지

- Booking.com / Agoda / Hotels.com / Expedia 등 OTA 사이트에서 다운로드 금지
- 호텔 공식 사이트 무단 다운로드 금지 (저작권은 사진작가에게 있는 경우 다수)
- 인스타그램·블로그·여행카페 사진 무단 사용 절대 금지

### 2-3. 호텔별 사진 확보 워크플로우

1. **1순위**: 호텔스컴바인 API hot-link (다운로드 X, IMG src로만)
2. **2순위 (대형 호텔)**: 호텔 공식 PR 키트 직접 요청 → 어드민에 "사진 사용 허락 받음" 체크
3. **3순위**: Wikimedia Commons에 호텔 사진 있으면 사용
4. **4순위**: 자체 OG 이미지 (호텔명+도시+평점) 자동 생성
5. **금지**: 위 4가지로 못 구하면 그 호텔은 추천 목록에서 제외

### 2-4. 메타데이터

모든 이미지는 다음 4개 필드를 함께 저장 (DMCA 대응):

```js
{
  url:            'https://...' (또는 /images/...),
  source:         'korea-tour-api' | 'wikimedia' | 'unsplash' | 'pexels' | 'hotelscombined-api' | 'self-og',
  license:        '공공누리 1유형' | 'CC BY-SA 4.0' | 'Unsplash License' | '파트너 약관' | '자체',
  credit:         '한국관광공사' | 'Photo by John Doe' | 'via HotelsCombined' | ...,
  sourceUrl:      'https://...',  // 원본 페이지
  alt:            '한라산 백록담',
}
```

## 3. 호텔스컴바인 어필리에이트 정책

- **단일 어필리에이트**: HotelsCombined CJ 또는 Travelpayouts 1개만 사용
- **딥링크 형식**: `https://www.hotelscombined.com/Hotel/{hotel_id}?aid={affiliate_id}&label={post_slug}`
- **`label` = 포스트 슬러그**: 어떤 페이지에서 클릭됐는지 추적
- **금지 표현**:
  - "최저가 보장" — 가격은 변동
  - "수수료 없음" — 일반화 금지
  - 호텔스컴바인 트레이드마크 키워드 입찰 (광고 운영 시)
- **고지 의무**: 모든 어필리에이트 링크 근처 + 푸터에 "본 사이트는 호텔스컴바인 제휴 활동의 일환으로 일정 수수료를 제공받습니다"
- **KPI**: 예약 전환율 3% 이상 유지 (어카운트 정지 리스크)

## 4. 표기 의무

- 호텔 가격: "OO년 O월 기준, 1박, 객실 타입 명시"
- 환율: "OO년 O월 OO일 기준"
- 거리: "직선거리 / 차량 / 도보 / 대중교통" 구분
- 항공 시간: "직항 / 경유 명시"
- 비자: "OO년 OO월 기준 — 출국 전 외교부 확인"

## 5. 의무 면책

- 모든 페이지 하단: `<TravelDisclaimer />` (비자·환율·호텔 가격 변동 안내)
- 비자/안전 페이지(Level A): 본문 상단 `RiskBox` 추가 — "출국 전 외교부 재확인 필수"
- 호텔 페이지: CTA 근처에 "쿠팡 파트너스 활동의 일환으로 일정 수수료를 제공받습니다" 또는 호텔스컴바인 고지
